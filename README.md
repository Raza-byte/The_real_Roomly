> An interior design application with a real-time 3D room builder — design, furnish, and preview custom spaces interactively.

---

## Features

- Real-time 3D room builder
- Customizable furniture and décor placement
- Live room preview
- Save, Export and revisit your designs

---

## Tech Stack

| Layer    | Technology |
|----------|------------|
| Frontend | React / Vite |
| Backend  | Node.js / Express |
| Database | MongoDB |

---

## Getting Started

### Prerequisites

- Node.js `>=18.x`
- MongoDB running locally (default port `27017`)

### Installation

```bash
# Clone the repo
git clone https://github.com/your-username/Roomly.git
cd Roomly
```

#### Backend

```bash
cd backend
npm install
npm run dev
```

#### Frontend

```bash
cd frontend
npm install
npm run dev
```

App runs at `http://localhost:5173` by default.
